# Part 1. Core Directories
## `/`(root) 
The top-level directory; every single file and folder starts here.

### Observed using `cd / && ls`
**Output sinppet:**
```
bin                etc                lib64       opt   sbin                sys
bin.usr-is-merged  home               lost+found  proc  sbin.usr-is-merged  tmp
boot               lib                media       root  snap                usr
dev                lib.usr-is-merged  mnt         run   srv                 var
```

## `/home`
User home directories with personal files/configs, each user gets their own workspace.
### Observed using `ls -l /home`
**Output sinppet:**
```
drwxr-x--- 5 ubuntu ubuntu 4096 Jan 29 11:35 ubuntu
```

## `/root`
The specific home directory for the "root" (Administrator) user. (not in /home for security).
### Observed using `sudo ls -l /root`
**Output sinppet:**
```
drwx------ 3 root root 4096 Jan 27 09:31 snap
```

## `bin` 
Contains essential command binaries needed for the system to boot and run in single-user mode (e.g., ls, cp).

## `/usr/bin`
Contains the majority of user commands and executable programs. (e.g., git,curl,vim).

## `/opt`
Third-party apps (manual installs like Google Chrome).
